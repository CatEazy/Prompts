# 腾讯云AI代码助手(已支持Craft模式相当于Cursor的Composer)
## OCR
```
写一个OCR文字识别应用
- 前后端网页应用
- Auth系统,JSON保存用户信息
- 登录用户可以保留文件,保留文件存在于服务端服务器根目录下的/saves/username/files
- 未登录用户限制保存3个文件,单个文件保存1分钟
- 前端框架React
- 后端Python
- 必要的文件(requirements.txt README.md等等)务必生成
- 请先生成plan.md然后根据plan.md生成项目
```
